# 2PJ.-Homelab-SIEM---Wazuh

**Цель:** Построить лабораторию SOC для практики мониторинга безопасности.

**Что сделано:**
- Развернут SIEM Wazuh 4.7 на Ubuntu Server 24.04
- Настроен сбор логов с двух хостов (Ubuntu + Simply Linux)
- Проверена работоспособность: агенты есть в списке и готовы к активации, логи поступают

**Стек:** VirtualBox, Ubuntu, Wazuh (Elastic Stack), Syslog

**Навыки:** Развертывание SIEM, настройка агентов, администрирование Elastic.

**Скриншоты:** Установка, дашборд, агенты.
